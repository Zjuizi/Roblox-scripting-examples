# Roblox Scripting Examples

Examples of Roblox Lua systems created by me.

## Included Systems

### RoundManager.lua

Round system with maps, gamemodes, scoring, XP, rewards and DataStore saving.

### EmoteBoxSystem.lua

Emote inventory, emote boxes, animations and inventory saving.

### PlayTimePassSystem.lua

Playtime reward progression system with milestones and daily resets.
